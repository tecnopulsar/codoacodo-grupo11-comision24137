Introducción

En este curso, aprenderás los fundamentos del desarrollo web, incluyendo HTML, CSS y JavaScript. Te prepararemos para crear sitios web interactivos y dinámicos.

Temas del Curso

  • HTML: Estructura de la web
  • CSS: Estilos y diseño
  • JavaScript: Interactividad
  • Control de versiones con Git y GitHub
  • Introducción a frameworks y bibliotecas
  • Trabajo con bases de datos
  • Despliegue de aplicaciones web

Proyectos

A lo largo del curso, trabajarás en varios proyectos prácticos que te ayudarán a consolidar tus conocimientos y a construir un portafolio que podrás mostrar a futuros empleadores.
